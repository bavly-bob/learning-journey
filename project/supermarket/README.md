# Supermarket / E-Commerce System (C++)

## Description
A modular console-based e-commerce system built as a team project.

## Features
- User authentication and roles (customer, special customer, admin)
- Product inventory management
- Shopping cart and checkout
- Wallet system with credit card validation (Luhn algorithm)
- Role-based discounts
- Persistent storage using files

## Architecture
- ProductDB
- UserDB
- CartDB
- EWalletDB

## Concepts Used
- Object-Oriented Design
- File I/O
- STL containers
- Input validation
- Team collaboration

## Notes
This project was developed collaboratively in a 4-person team.
